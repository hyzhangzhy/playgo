# ST 表 (稀疏表)

稀疏表，又称倍增表，通过倍增的方法构建一张只读的表，来实现 O(1) 或者 O(logn) 的区间查询。

本质为动态规划（倍增思想）。定义 $st[i][p]$ 表示从位置 $i$ 开始，向后跳 $2^p$ 步到达的位置（或状态）。

状态转移方程：
$$
st[i][p] = st[st[i][p-1]][p-1]
$$

**公式含义：**
想要跳 $2^p$ 步，可以拆分为两步：
1. 先跳 $2^{p-1}$ 步（即前一半路程），到达中转点 $mid = st[i][p-1]$。
2. 再从中转点 $mid$ 跳 $2^{p-1}$ 步（即后一半路程），到达终点。
